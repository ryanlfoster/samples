About MediaFINDER

What is MediaFINDER?

MediaFINDER is a web application that helps you locate Church media files (Kathryn, I added this because at
some point it will also offer video and audio files; I don't want to forget to change the wording at that point.)
for use in your Church employment. MediaFINDER specifies the Intellectual Property Code for each file. See the
Rights Management section below.

Who can use MediaFINDER?

MediaFINDER is available to the entire Church workforce and must be accessed from the Church network. This
may require a Virtual Private Network (VPN) connection at some locations.

Can I use MediaFINDER on my tablet or smartphone?

MediaFINDER was designed with both tablets and PC's in mind. Use on a smartphone is possible but not
recommended.

MediaFINDER Images

Where do MediaFINDER images come from?

Media files can come from a variety of source. Right now Telescope is the primary source for images in
MediaFINDER. The source repository is shown in Details View under Summary.

Can I submit an image to MediaFINDER?

Yes, by submitting them to Telescope. Check with the person in your department who is able to submit images to
Telescope.

What if I have a media repository that I would like MediaFINDER to search?

Please contact the Product Manager (James Hooper: james@ldschurch.org) if you would like to make additional
media files searchable through MediaFINDER.
